# Career Guidance

A React + Vite landing page for career guidance and roadmap planning.

## Features
- Bootstrap-based responsive UI
- Interactive roadmap builder
- Progress tracking for roadmap items
- Contact and FAQ sections

## Run locally
```bash
npm install
npm run dev
```

## Build for production
```bash
npm run build
```

## Deploy
The production build is generated in the `dist/` folder and can be deployed to Netlify, Vercel, GitHub Pages, or any static host.
